# 🏥 Healthcare Data Analysis (Realistic Dataset)

## Overview

This project analyzes patient and hospital visit data using realistic full-name datasets. It explores trends in hospital operations, patient behavior, and revenue generation.

## Dataset

* 30+ patients (realistic names)
* 80+ hospital visits
* Multiple departments and diagnoses

## Key Insights

* Identified high-value patients
* Cardiology generated the most revenue
* Repeat visits indicate chronic conditions
* Revenue trends vary by date and city

## Skills Demonstrated

* SQL joins and aggregations
* Data modeling with foreign keys
* Data visualization in Excel
* End-to-end data analysis workflow
